# Changelog

All notable changes to OrionClaw.

## [1.0.0] - 2026-03-26

### Added
- Initial release based on OpenClaw
- One-click installer for Windows (WSL2), macOS, and Linux
- Multi-agent support with cost warnings
- Free provider integrations:
  - Modal GLM-5 (free until Apr 30, 2026)
  - OpenRouter
  - Google AI
  - Ollama (local)
  - OpenCode
  - Kilo Gateway
- Paid provider integrations:
  - Anthropic
  - OpenAI
  - GitHub Copilot
- Channel support:
  - Telegram
  - Discord
  - WhatsApp
  - CLI
- Workspace templates:
  - AGENTS.md
  - SOUL.md
  - USER.md
  - IDENTITY.md
  - MEMORY.md
  - BOOTSTRAP.md
- Documentation:
  - Skill creation guide
  - Multi-agent setup guide

### Security
- Removed Kali-style security tools (too dangerous for general distribution)
- Added safety guidelines in AGENTS.md

Based on [OpenClaw](https://github.com/openclaw/openclaw)
