# BOOTSTRAP.md - First Run Setup

_Welcome! Let's set up your agent._

## The Conversation

When you first start, have a conversation with your agent:

1. **Name your agent** — What should it be called?
2. **Define personality** — Formal? Casual? Witty?
3. **Set boundaries** — What should it never do?
4. **Configure channels** — Telegram? Discord? CLI only?

## After Setup

Update these files with your preferences:

- `IDENTITY.md` — Agent's name, personality, emoji
- `USER.md` — Your name, preferences, timezone
- `SOUL.md` — Detailed personality traits

## Delete This File

Once setup is complete, delete this file. You won't need it again.

_Good luck! Make your agent yours._
